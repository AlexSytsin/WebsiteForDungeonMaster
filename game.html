Немного об игре

Описание игры

- Исследуйте Темные Подземелья: Каждый уровень полон тайн и секретов, ждущих быть обнаруженными.

- Сражайтесь с Врагами: От орков до драконов, каждый враг требует особого подхода и стратегии.

- Спасите Принцессу: Ваша главная миссия - спасти принцессу. Будьте готовы к испытаниям!

- Улучшайте Своего Героя: Собирайте добычу и улучшайте свои навыки и оружие.


Технологии и языки программирования, использованные при написании проекта:

    • Игра:
      1. GitHub
        • Совместная разработка
        • Контроль версий
      2. Unity
        • Физика объектов и персонажа
        • Анимации движения и стрельбы
      3. C#
        • Логика взаимодействия объектов
        • Игровые механики
    • Сайт:
      1. HTML
        • Создание макетов страниц и расположение элементов
      2. CSS
        • Стили и оформление
      3. JavaScript
        • Адаптация дизайна и функционала под разные типы устройств
        • Загрузка ссылок
        • Скачивание игры
      4. WebGL
        • Возможность запускать игру прямо в браузере
      5. GitHub Pages
        • Бесплатная публикация сайта в интернете с кастомным доменом
